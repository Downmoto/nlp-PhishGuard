[abstract]
This report presents PhishGuard, a phishing email detection system built by fine-tuning `bert-base-uncased` for binary classification. The project combines data preparation, preprocessing, model training, evaluation, explainable inference, and a Gradio-based web interface into one reproducible pipeline. On the held-out test split, the final model achieved 99.48% accuracy, 0.9948 macro F1, and 0.9998 ROC-AUC. The result shows that a compact, well-engineered transformer pipeline can distinguish legitimate email from phishing attempts with very high reliability while remaining practical to deploy and inspect.
